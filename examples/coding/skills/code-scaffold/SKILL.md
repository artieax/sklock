# Code Scaffold

Generate boilerplate for a new module, React component, API route, or service
from built-in templates. No dependencies, no sub-skills — fully self-contained.

## Inputs

- `type` — `module` | `component` | `route` | `service`
- `name` — identifier for the generated artifact
- `output_dir` — directory to write files into

## Outputs

- Generated files written to `output_dir`
