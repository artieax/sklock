# Citation Formatter

Format citations in APA, MLA, or Chicago style given a source URL or metadata object.

## Inputs

- `source` — URL or `{ title, author, year, publisher }` object
- `style` — one of `apa` | `mla` | `chicago` (default: `apa`)

## Outputs

- Formatted citation string
